# Assets

Placeholders à fournir avant build EAS :

- `icon.png` — 1024×1024 (icône app)
- `adaptive-icon.png` — 1024×1024 (Android adaptive)
- `splash.png` — 2208×1242 ou supérieur, fond `#0b0f14`
- `tv-banner.png` — 320×180 (Fire TV banner, requis pour Android TV)
- `favicon.png` — 32×32 (web fallback, non utilisé sur TV)
- `fonts/Inter-*.ttf`, `fonts/Manrope-*.ttf` — chargées via `expo-font`
